    # 時間4：2020.03.19

    有些道理，不必等到完全掌握了才去执行，例如：冥想、要事优先、少吃糖，每天花一整块时间思考……

    “Skin in the Game",你想做成一件事，你必须要来真的，”流血“是不可避免的。

    前几天，在宿舍办公，每天8点钟起床，现在，去公司上班，6:30起床，在公司戴8小时口罩，会宿舍，一点斗志都没了，很疲惫，开始长时间刷抖音了，把时间浪费在这上面，毫无意义！毫无意义！毫无意义！

    还有，每天时不时的查看疫情有关的新闻，毫无意义，并且对自己的身体有害。

    做对的事情，而不是图一时安乐，你要斗争啊！你要斗争啊！你要斗争啊！

    做对的事情，如果感到进步太慢，请保持足够的耐心，快速入门很重要，把问题是什么弄明白很重要。

    和你，共同承担风险。

    有些人，很坏，若只是坏，那就从物理上远离ta……

    曾经的”兔子“，如演化的足够快，最终成了"狮子"，ta可以主动选择吃还是不是这只狐狸。

    # 時間3：2020.02.08

    早在1942年，哈佛医学院的生理学系主任沃特尔·坎农(Walter Bradford Cannon)就注意到一个现象，被他称之为“巫毒死”（Voodoo Death）。

    坎农教授亲自处理过一个病例。一位年轻人在部落里被巫师用骨头指了一下之后当场病倒，卧床不起——不是装病，是真的各方面指标都显示确实病了，并且越来越严重……随后有一天巫师前来告诉他，当时用骨头指的并不是他的时候，这位年轻人的所有病症几乎当场全部消失了。

    2019年12月1日到现在，新冠肺炎已经蔓延到很多地方了，在疫情地图上显示最近的感染点与我的距离是319m,这几天，由于口罩的不足，基本上没有出门，出门也是在小区里面的超市买食物。

    2020年，最大的愿望是能够健康的活下来，在灾难来临时，才感到生命的宝贵，如果疫情结束，我最想干的一件事就是去小肥羊餐厅吃一次火锅，虽然很贵，但是很好吃。

    2020年，一个黑天鹅事件就把所有人打的措手不及，一定会有很多企业面临倒闭的风险，对于个人，没有足够的储蓄，生活也是灾难性的，资金链断裂是非常可怕的事情。

    在灾难面前，手上有6个月的工资储备是非常重要的，如果你现在的钱不足以让你6个月白吃白喝，那现在的你是需要加倍努力的。

    生物演化，会淘汰掉哪些动物？物种间，会淘汰掉那些弱小的生物，

    那人呢？这几天，武汉的新肺炎让很多人死去，因为没有疫苗，没有特效药，检测试剂有时还会检测错误，出现假阴性，唯一能够让病人活下去的，就是自身的抵抗力，所以，提高抵抗力是非常重要的，要经常吃高蛋白的食物，要加强锻炼，要减肥:）

    2020年2月份，我想把李笑来老师的书重新读一遍，他的很多书都开源了，在电脑上看李笑来老师的书，字体太小了，我用插件改变了文字的大小和字体的样式，看上去就好看多了，我想摘抄李笑来老师的一段话，是在《把时间当做朋友》里看到的：

    首先要解决的是睡眠问题，必须达到7个小时的睡眠，

    策略是：中午1点左右和下午18点左右(下班回来后)，打盹30分钟。

    这样就有了1个小时的时间，那晚上再睡6个小时就可以了。

    实践这个方案的一个小窍门是：

    在打盹之前喝一小杯咖啡，然后迅速睡过去。这样，在30分钟后醒来时，大脑就会处于清醒的状态，这种感觉真的很棒。

    按时打盹，按时清醒，是需要锻炼的。我差不多花了两个月的时间，才学会了这种本事。

    所以，我决定去淘宝上买一些无糖咖啡，但是，我这里还有几包含糖咖啡，先喝完再买吧。

    # 時間2：2019.8.16

    很多有价值的东西，需要不断反复的学习才能掌握，好东西就应该得到它，哪怕付出很多的时间与精力。

    今天，我想写的是耐心，耐心真的很重要，一步登天是不可能的，我自己必须时刻告诉自己，必须一步一步的向前走，我要学的东西有很多，很多都在我的计划之内，心理学和编程是我最近要学的，Java编程是与工作相关的，必须好好学习。

    为什么要学心理学？因为我想了解我自己，或许童年时的某些经历对我造成了一定的阴影，导致我对某些事情有着错误的观点，模糊不清的恐惧才是真正的恐惧，我需要通过读心理学的书籍来正确的解读曾经遇到过的事情，很多事情需要重新评估。

    有些事情是不应该公之于众的，所以，博客应该是整理版，就是在本地写好了，把隐私的事情去掉。

    想学好编程，需要不断的积累。Java、Python、ES6、VUE，等等，都需要学，算法也需要学，甚至英语也需要学，时间有限，我知道未来一定比现在美好，刚开始进步的曲线很平缓是很正常的，等到了一定的时候，等知识都积累够了，进步是会更快一些的，应该是指数级的增长。
